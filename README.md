# Task Manager

A simple web-based Task Manager built with Flask. You can add, complete, and delete tasks.

## Features
- Add new tasks
- Mark tasks as complete or incomplete
- Delete tasks
- Lightweight and easy to run locally

## Tech Stack
- Python 3
- Flask
- SQLite
- HTML/CSS

## Setup and Run
